当日价格紧凑版式草案

目标是把当前偏卡片墙的空间占用压下来，同时尽量保留“大盘 vs 个股”分组感。下面 3 个方案都基于同一组 mock 数据，重点比较密度、扫读效率和视觉秩序。

方案 A：紧凑横排行

保留卡片感，但每个卡片压成一行，最接近你现有设计，改造成本最低。
