# 测试环境说明文档

## 环境划分

| 环境 | 用途 | 绑定分支 | APP_ENV |
|------|------|----------|---------|
| 本地 | 个人开发、快速验证 | 无 | test |
| 开发/测试 | 联调、集成测试 | 非 master | test |
| 生产 | 线上服务 | master | prod |

## 资源清单
> 按项目实际情况填写

| 资源类型 | 测试资源 | 生产资源 |
|----------|----------|----------|
| 数据库   |          |          |
| 服务实例 |          |          |
| 前端入口 |          |          |

## 环境变量
参考根目录 `.env.example`，必须包含 `APP_ENV` 标识。

## 健康检查
健康检查接口应返回：
- 当前 `APP_ENV` 值
- 服务版本
- 关键依赖连通性状态
