## BUG-1: 关键词 Tab 点击被 tooltip 遮挡 [已修复]

**严重程度**: 高

**复现步骤**:
1. 打开关键词管理页面
2. 尝试点击"噪音"Tab

**预期行为**: Tab 切换成功

**实际行为**: 点击被标题栏的 `.bubble` tooltip 元素拦截，导致点击失败

**根因**: `.tip-trigger .bubble` 设置了 `pointer-events: auto`，即使 `opacity: 0` 时也会拦截点击事件

**修复方案**:
```css
.tip-trigger .bubble {
  pointer-events: none;
  visibility: hidden;
  /* ... */
}

.tip-trigger:hover .bubble {
  pointer-events: auto;
  visibility: visible;
}
```

**状态**: ✅ 已修复 (2026-03-22)
